Performance and Customization

The UI is rendered on the GPU, offloading the CPU for the crucial part: Adio processing. The plugin is freely resizable and highly customizable. Themes and other modifications can easily be done via convenient `json` configuration files.

Plugin formats

  • Windows 10 or higher: VST3, CLAP
  • Ubuntu 22.04: VST3, CLAP
  • macOS: AU, VST3, CLAP

System requirements

  • 64 bit only
  • OpenGL 3 or higher
