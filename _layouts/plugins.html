Performance and Customization

Rendering the responsive UI is performed by the GPU, offloading the CPU for the crucial part: Audio processing. The plugin is freely resizable and highly customizable. Themes and parameter modifications can be done via json configuration files. All Darkpalace Studio plugins offer various themes.

Plugin formats

  • Windows 10: VST3, CLAP
  • Ubuntu 22.04: VST3, CLAP
  • macOS 12, 13, 14 (Intel and Apple Silicon): AU, VST3, CLAP

System requirements

  • 64 bit only
  • OpenGL 3 or higher
Note: macOS Plugins are not signed. Thus on mac you will need to execute a short command in the shell prior to using it. This is described in the FAQ or the respective manual.
